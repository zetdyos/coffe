
        Start with any file in folder 'demos', 'examples'
        e.g. you can start with 'examples/simple-slider.source.html'.
        Use 'jssor.slider.debug.js' for debug.
        Use 'jssor.slider.mini.js' or 'jssor.slider.min.js' for release.
        In addition you can run 'examples/simple-slider.compress.bat' to compress your slider to copy/paste code snippet, see 'examples/simple-slider.compress.bat'.


        http://www.jssor.com/help
        http://www.jssor.com/tutorial
        http://www.jssor.com/development


        Note that 'jssor.compress.exe' runs on windows system with .NET 2.0 installed.
